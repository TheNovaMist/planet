# 画一个彩色星球

 * 目标：实现一个图形化的界面并在其画板上画出可改变颜色的小星球
 * 具体步骤：
   * GUI基本设置
   * 在继承JPanel的Draw类中画图
   * 使用Timer令图画不断切换
   * 按钮绑定鼠标事件监听通过点击控制画图切换的开始与停止
